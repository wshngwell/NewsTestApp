## При написании данного тестового задания использовались слудующие технологии
1. Kotlin.
2. Kotlin Coroutines.
3. Clean Architecture.
4. MVVM (Model-View-ViewModel) архитектура.
5. Использование Room для сохранения новостей в базу данных.
